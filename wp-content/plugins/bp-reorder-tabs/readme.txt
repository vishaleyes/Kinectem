Easily reorder BuddyPress Profile and Group tabs from the WordPress admin.

== Description ==

Easily reorder the tabs (navigation) and sub-tabs on BuddyPress Profiles and Groups, and set any tab as the default one users will land on. You can even reorder tabs added via plugins.

== Installation ==

1. Make sure BuddyPress is activated.
2. Visit 'Plugins > Add New'
3. Click 'Upload Plugin'
4. Upload the file 'bp-reorder-tabs.zip'
5. Activate BuddyPress Reorder Tabs from your Plugins page.

= Configuration =

1. Visit 'Settings > BP Reorder Tabs' and select your desired options.
